# Contributing

Since this repo holds a Kibana plugin, you'll first want to get familiar with how Kibana as a whole works by reading [the Kibana section](#kibana).
If you're already familiar with Kibana, you may jump straight to [the External Plugins section](#external-plugins).

## Kibana

To get up to speed on contributing to Kibana, see the [official Kibana contributing guidelines](https://github.com/elastic/kibana/blob/main/CONTRIBUTING.md).

## External Plugins

Kibana has a number of built-in plugins that live alongside the source code for the platform itself, but this repo contains an _external_ plugin. While the two may be indistinguishable to end users, the development workflow differs. See the [official guide for external plugin development](https://www.elastic.co/docs/extend/kibana/external-plugin-development).

## Getting started

For a step-by-step guide to setting up a local development environment (cloning Kibana, linking this plugin, starting Elasticsearch, loading sample data, and running Kibana with the plugin), see [Quick Start (Development)](README.md#quick-start-development) in the README.
